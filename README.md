# 📚 KITABU - Plateforme de Location de Livres Académiques

**KITABU** ("livre" en swahili) est une solution web conçue pour faciliter l'accès aux ouvrages académiques pour les étudiants de l'**Université Protestante au Congo (UPC)**. Le projet vise à rendre le savoir plus accessible en proposant un système de réservation en ligne simple et efficace.

---

## 🚀 Fonctionnalités

* **Catalogue Dynamique** : Visualisation d'une large sélection d'ouvrages couvrant les mathématiques, l'algorithmique, le droit et l'économie.
* **Recherche Intuitive** : Filtrage en temps réel des livres par titre via une barre de recherche dédiée.
* **Système de Réservation** : Formulaire complet permettant de choisir un ouvrage, un point de retrait et une date précise.
* **Points de Retrait Locaux** : Informations détaillées sur les lieux de retrait, incluant la Bibliothèque de l'UPC et le Bureau du comité d'étudiants.
* **Interface Responsive** : Design moderne et épuré avec une navigation fluide adaptée aux besoins académiques.

## 🛠️ Technologies Utilisées

* **Frontend** : HTML5, CSS3 (Variables CSS, Grid, Flexbox).
* **Scripting** : JavaScript (Vanilla JS) pour la gestion dynamique du catalogue et des interactions utilisateurs.
* **Design & Ressources** : 
    * *Google Fonts* : Poppins & Playfair Display.
    * *Font Awesome* : Bibliothèque d'icônes pour l'interface.

## 📂 Structure du Projet

```text
├── index.html          # Page d'accueil et présentation de la vision
├── catalogue.html      # Interface de recherche et catalogue de livres
├── points.html         # Localisation et horaires des points de retrait
├── styles.css          # Feuilles de style globales et design system
├── script.js           # Logique JS (données locales, filtres, réservations)
└── Images/             # Dossier contenant les couvertures des ouvrages

```

## ⚙️ Installation et Utilisation

1. **Cloner le dépôt** :
```bash
git clone [https://github.com/akalefa-27/kitabu.git](https://github.com/akalefa-27/kitabu.git)

```


2. **Lancer l'application** :
Il suffit d'ouvrir le fichier `index.html` dans n'importe quel navigateur moderne. Aucune configuration backend n'est requise pour cette version (données gérées en local via JavaScript).

## 📝 À propos de KITABU

Le projet repose sur une vision claire : **Remettre le livre au cœur de l'apprentissage !**

> "KITABU est bien plus qu'une simple plateforme de location. C'est une solution locale pensée pour permettre à chaque étudiant d'avoir un accès rapide, abordable et flexible aux ouvrages essentiels à sa réussite."
